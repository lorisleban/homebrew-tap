# Lorisleban Tap

## How do I install these formulae?

`brew install lorisleban/tap/<formula>`

Or `brew tap lorisleban/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lorisleban/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
